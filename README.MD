# UrlShortCut
## О проекте:
Чтобы обеспечить безопасность пользователей, все ссылки на сайте заменяются ссылками на наш сервис.
Сервис работает через REST API.

### Технологии

__Java17__
__Maven 3.8__
__PostgreSQL__
__Spring boot__
__Spring security__
__Liquibase__
